# <img src="icons/icon.png" height=64px width=64px></img> Password Hasher</br>
Simple password hasher extension for chrome.<br>
Add's domain name to your password, hashes it using SHA256, </br>
then takes first 20 characters and adds "B.2" to end and puts it in a password field.
